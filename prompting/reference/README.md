## üìù Quick Tips

### Effective Prompting
1. **Be specific**: Clear, unambiguous instructions
2. **Provide context**: Relevant background information
3. **Set constraints**: Define boundaries and requirements
4. **Specify format**: How you want the output
5. **Use examples**: Show what you want

### Common Patterns
- **Role + Task**: "You are a {role}. {task}"
- **Step-by-step**: "Step 1: ... Step 2: ..."
- **Example-driven**: "Example 1: ... Example 2: ..."
- **Structured**: "Format: {format}"
